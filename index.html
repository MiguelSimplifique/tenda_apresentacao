Segurança & LGPD

Construído para
ambientes corporativos.

Todos os dados da Tenda pertencem 100% à Tenda. Exportação irrestrita a qualquer momento.

Proteção dos Dados

  • Em trânsito: TLS 1.3 obrigatório
  • Em repouso: AES-256 (Supabase nativo)
  • Senhas: bcrypt via Supabase Auth
  • Banco no Brasil: AWS São Paulo

Controle de Acesso

  • MFA obrigatório: Admin e Gestor (Google Authenticator TOTP)
  • MFA opcional: Vendedores
  • Row Level Security: cada vendedor vê só seus leads
  • Perfis: Admin / Gestor / Vendedor
  • Logs de auditoria — 12 meses de retenção

Conformidade LGPD

  • Contrato DPA com OpenAI e Anthropic
  • Política de incidentes documentada
  • Dados processados por IA não usados para treinamento
  • Retenção configurável; exclusão a qualquer momento

Segregação de Dados

  • Isolamento lógico por cliente (single-tenant logic)
  • Banco dedicado por cliente no Supabase
  • Variáveis de ambiente separadas por projeto
  • Firewall dedicado por servidor
Supabase — SOC 2  ·  Hetzner — ISO 27001  ·  Vercel — SOC 2  ·  OpenAI / Anthropic — DPA LGPD  ·  Meta — WhatsApp Business

📊 Como funciona o Score
Pipeline de 3 etapas: conversa → análise → classificação
💬

1. IA Conversacional coleta os dados

A primeira IA conduz a conversa e coleta: imóvel desejado, renda familiar, FGTS disponível, urgência de compra, localização, situação de crédito. Tudo armazenado no banco de dados em tempo real.

🧠

2. IA de Análise processa e pontua

Ao encerrar a conversa, uma segunda IA analisa todos os dados coletados considerando: adequação de renda ao produto, capital disponível (FGTS), urgência de compra e engajamento na conversa.

Renda x ProdutoPeso 35%
Urgência de CompraPeso 30%
FGTS / CapitalPeso 20%
EngajamentoPeso 15%
🎯

3. Classificação automática e entrega

Score 0–100 calculado e lead classificado. Dados salvos no Supabase e painel do vendedor atualizado instantaneamente.

🔥 Quente — Score >70 — Notificação imediata 🟡 Morno — Score 40–70 — Visível no painel ❄ Frio — Score <40 — Registrado para análise
